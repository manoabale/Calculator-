# Calculator-🔢 

A simple and responsive Calculator built with **HTML**, **CSS**, and **JavaScript**.  
This calculator performs basic arithmetic operations and is perfect for beginners learning DOM manipulation.



## 🚀 Features

- Addition, subtraction, multiplication, division  
- Clean and modern UI  
- Fully responsive layout  
- Works on CodePen, GitHub Pages, Netlify, and any static hosting  
- Beginner‑friendly code structure  



## 📂 Project Structure



calculator-app/ │ ├── index.html ├── style.css └── app.js




## 💻 Technologies Used

- **HTML5** – Structure  
- **CSS3** – Styling  
- **JavaScript (Vanilla)** – Logic and button interactions  

No frameworks required.
